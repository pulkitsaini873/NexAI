## ✨ Features

- **Multi-Model Orchestration:** Chat effortlessly with Llama 3 (via Groq), Gemini 1.5 Pro, Claude 3 (via OpenRouter), and more.
- **Ethereal Glassmorphism UI:** A completely distinct, premium aesthetic featuring animated aurora gradients, floating magnetic docks, and luxury typography (Playfair Display mixed with JetBrains Mono).
- **Web Search Synthesis:** Seamless DuckDuckGo integration contextually searches the web to provide real-time, grounded AI responses.
- **Image Generation:** Instantly synthesize concepts into visualizations using NVIDIA NIM integration.
- **Text-To-Speech (TTS):** Premium neural voice feedback with localized playback synchronization.
- **Local-First Privacy:** All API keys, chat history, and system configurations are stored locally on your device (`localStorage`). Nothing goes to a middleman server.
- **Context Optimization (Upcoming):** Hooked into NLP-based _Caveman Compression_ using Python `spaCy` to reduce input tokens locally without losing semantic meaning.

## 🛠 Tech Stack

- **Framework:** React 18 & Vite
- **Styling:** Custom CSS Matrix (Zero-dependency Ethereal Glassmorphism design system)
- **State & Storage:** React Hooks + `localStorage` adapters
- **Markdown Rendering:** `marked` + `highlight.js`

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+ recommended)
- Your own API keys (Groq, Google Studio, OpenRouter, etc.)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pulkitsaini873/NexAI.git
   cd NexAI
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Onboarding:**
   Open the application in your browser (default `http://localhost:5000`). The initial launch will prompt you softly for your API Keys via a secure, local-only modal.

## 🎨 Aesthetics & Design
NexAI distances itself from standard "AI chatbot" looks. We utilize deep obsidian backgrounds, translucent floating panels (`backdrop-filter: blur(24px)`), and a mix of Serif (Playfair Display) for editorial storytelling and Monospace (JetBrains Mono) for technical delivery.
